## TITLE - Netflix Movie Recommendation System
## INTRO - 
This is a Netflix-style Movie Recommendation System built using Machine Learning and Streamlit. 
The system recommends similar movies based on content similarity using NLP techniques such as Count Vectorization and Cosine Similarity.
Movie posters are fetched dynamically using the TMDB API.

## FEATURES 
- Netflix-style dark UI
- Movie recommendation engine
- TMDB movie posters
- Content-based filtering
- Cosine similarity algorithm
- Interactive Streamlit web app

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- TMDB API
- Pickle
